# Return Unique Value

Restful API returning unique value where input >= unique output

## Usage

Pointing folder

	cd foldername

after folder pointed, start service

	composer start

enter url in your web browser
	
	http://host:port/api/get?number=[enter_number]
	

## Unit Testing

	.\vendor\bin\phpunit
